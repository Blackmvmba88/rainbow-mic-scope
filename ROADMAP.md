# Roadmap

Rainbow Mic Scope is moving from visual experiment to usable creative tool. The product goal is simple: make live microphone visuals easy to capture, record, save, and share.

## v1.5 - Capture And Presets

- Add PNG canvas capture.
- Add WebM recording with `MediaRecorder`.
- Add fullscreen clean mode.
- Add built-in preset registry.
- Save and load presets from `localStorage`.
- Add import/export preset JSON.
- Add shareable URL configuration.

## v1.6 - Audio And Demo Modes

- Add microphone/input selector to the stable WebUI.
- Add no-mic demo mode with oscillator/noise input.
- Add browser permission error UI.
- Add gain normalization safeguards.
- Add signal diagnostics panel.

## v1.7 - Public Demo Polish

- Add a landing/demo page for GitHub Pages.
- Add screenshot gallery.
- Improve mobile layout.
- Add help/about modal.
- Add public preset examples for social visuals.

## v1.8 - Repo And Release System

- Add GitHub Actions static smoke checks.
- Add semantic release checklist.
- Add release artifact workflow.
- Add optional static validation scripts.
- Add version badge and published demo badge.

## Later

- Optional desktop wrapper with Tauri or Electron.
- MIDI/control surface support.
- Audio file playback mode.
- Transparent-background export path.
- Advanced color mappings by pitch, centroid, and spectral bands.
